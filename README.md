# Что такое шаблоны ООП

Шаблоны ООП — это подходы, которые позволяют строить правильную архитектуру программ.  
В данной работе задействованы:

* **Интерфейсы** — задают контракт поведения (`Fractionable`, `Meowable`);
* **Инкапсуляция** — скрытие полей классов (`Fraction`, `Cat`);
* **Полиморфизм** — возможность передавать разные реализации (`MeowCounter`);
* **Переопределение методов** — `toString()`, `equals()`;
* **Generics (дженерики)** — используются в списках и методах.

---

# Что такое коллекции

Коллекции Java позволяют работать с наборами данных:

* **List** — список элементов;
* **Set** — множество уникальных значений;
* **Queue** — очередь FIFO;
* **Map** — словарь ключ–значение.
